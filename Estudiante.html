C#

C# es un lenguaje de programación de propósito general desarrollado por Microsoft y lanzado en 2000. Se usa para una variedad de tareas, como aplicaciones web, aplicaciones de escritorio, aplicaciones móviles y juegos. C# se diseñó como un lenguaje similar a C que utiliza .Net Framework de Microsoft. La sintaxis principal es similar a C, C++, Java.

HTML

HTML, abreviatura de HyperText Markup Language , es la base de todas las páginas web. Fue creado por Tim Berners-Lee en 1993 para definir la estructura de una página web. Sin él, no podría organizar el texto ni agregar imágenes o videos a sus páginas. HTML es el comienzo de todo lo que necesitas saber para hacer tu primer sitio web.

CSS

CSS, abreviatura de hojas de estilo en cascada, es un lenguaje de hojas de estilo que se utiliza para diseñar sitios web. Los colores, las fuentes y los diseños de página de un sitio se pueden administrar mediante CSS. Cuanto más cómodo se sienta con CSS, mejor equipado estará para crear un sitio web atractivo.
